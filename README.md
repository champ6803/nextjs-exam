## Description

[Next](https://github.com/vercel/next.js) The React Framework. Contribute to vercel/next.js development by creating an account on GitHub.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run dev

# production mode
$ npm run start
```
